Attacks on Application

In this project, the implementation of AWS WAF and a honeypot link aims to defend against three primary attacks: SQL injection, Cross-Site Scripting (XSS), and brute force attacks. AWS WAF filters and blocks malicious requests, effectively mitigating SQL injection attempts by analyzing patterns in the honeypot link's access. It also guards against XSS by identifying and blocking suspicious payloads. Additionally, the honeypot aids in monitoring brute force attacks, allowing for the logging of repeated attempts and enabling rate limiting to restrict malicious IP addresses. Together, these measures enhance the overall security of the web application.
